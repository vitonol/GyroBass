<h1 align="center">GyroBass 🥙🎸</h1>
<h3 align="center" >A real-time digital signal processing effect module an STM32 microcontroller using Daisy Seed. </h3>
<br>

## Overview

This pedal-like effects module is designed to be integrated into the body of a bass guitar/guitar. It features an LSM6DS3 gyroscope, which is mapped to dynamically control the filter cutoff frequency, responding intuitively to the player's movements.

It augments expressions while playing the instrument, allowing the player to control various sound parameters by using their body, transferring the instrument's tilt and rotation.

While main effect is being distortion, this project usese:

- State Variable Filter mapped to a button to switch between Low-Pass, Notch and Band-Pass modes.
